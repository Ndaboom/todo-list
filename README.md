## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- PC with a browser and Internet
- [Node.js](https://nodejs.org/en/download) installed
### Setup
- git clone [repo](https://github.com/Ndaboom/todo-list.git)
- cd todo-list
- code .
- run npm install
- run npm start
